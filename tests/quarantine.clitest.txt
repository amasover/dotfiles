--- bootstrap --probe-provider: substitution printed, self-resolution silent ---

The stub pacman answers like the real one: a provider name for flashfocus,
the same name back for selfsame, and "target not found" + exit 1 for
anything AUR-only. The AUR-only name goes FIRST: the 2026-07-15 VM run died
on exactly that (set -e/pipefail killed bootstrap on pacman's nonzero exit
before any later name was probed).

$ stub=$(mktemp -d) && printf '#!/bin/sh\nfor a; do :; done\ncase "$a" in\n  flashfocus) echo flashfocus-git ;;\n  selfsame) echo selfsame ;;\n  *) echo "error: target not found: $a" >&2; exit 1 ;;\nesac\n' > "$stub/pacman" && chmod +x "$stub/pacman" && PATH="$stub:$PATH" .local/bin/setup/bootstrap --probe-provider aur-only-pkg flashfocus selfsame > "$stub/out"; echo "rc=$?"
rc=0
$ tr '\t' ' ' < "$stub/out"
flashfocus flashfocus-git
$

--- bootstrap --probe-provider: unresolvable name alone is a clean no-op (the 2.33 regression) ---

$ PATH="$stub:$PATH" .local/bin/setup/bootstrap --probe-provider aur-only-pkg; echo "rc=$?"
rc=0
$
